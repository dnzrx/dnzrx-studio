## Overview

A repo of a site for **mixing & mastering commissions** under **dnzrx.studio | Imposter Syndrome Sound**.

Built as a static site:

- HTML partials for sections (intro, commission, flow, T&C, gallery)
- Modular CSS and vanilla JS (includes + render helpers)
- Brand and social icons as SVG assets

Deploy anywhere that serves static files; no compile step.

## Stack

- HTML / CSS / vanilla JavaScript
- Tailwind via CDN for utilities
